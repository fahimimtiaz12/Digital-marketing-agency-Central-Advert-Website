# Central-Advert-Website
Modern digital marketing agency website for Central Advert. Features include interactive payment demo, coupon code support, responsive design, and professional sections for services, portfolio, and contact.

This repository contains the complete source code for Central Advert’s modern digital marketing agency website. The site showcases our services, portfolio, client reviews, and includes interactive tools to engage visitors and drive conversions.

Overview:
Central Advert’s website is designed to help businesses succeed online by featuring:

Professional Design: Clean, responsive layouts optimized for mobile and desktop.

Interactive Components: Includes an animated payment demo page, coupon code discounts, galleries, and testimonial sliders.

Comprehensive Service Listings: Highlighting our expertise in SEO, social media advertising, branding, consulting, and more.

User Engagement Features: Contact forms, counters, typewriter hero text, and smooth scrolling for excellent user experience.

Features:
Full-featured payment demo form with card selection, coupon system, and animated feedback modal.

Coupon code integration for instant savings and user promotions.

Multiple service sections, media grid, client testimonials, and image gallery.

Local storage for contact form convenience and seamless UX.

Animated backgrounds, particles, and tilt/3D hover for visual impact.

Easy navigation and scroll-spy active highlighting.

Responsive mobile performance.

Tech Stack:
HTML5, CSS3, JavaScript (Vanilla)

Swiper.js for client review slider

AOS (Animate On Scroll) for reveal animations

No external frameworks required
